Creative projects

These are the projects that involve cultivating a piece of myself outside of myself.

All the these projects should follow these guidelines:

  1. It begins with a place in version control!
    • For strictly writing projects it can just be a part of my org files!
    • For anything else, it’ll get its own repository and get its own mirror on GitHub!
      • If it’s a single media file, it can go in a repository of single media files (like art repo, or music repo, etc)
  2. It should be produced with “literacy” throughout! (I.e., literate programming.)
  3. It should be produced according to whatever applicable guidelines I have set for my self.
  4. The end product should be distributed through appropriate channels! (Choose applicable ones:)
    • Tator Tot Club sub-site
    • GitHub releases
    • AUR
    • Language-specific repositories
    • App stores
    • Bandcamp
    • YouTube
    • Itch.io
    • Ko-fi post
  5. The end meta-product (literate product; product about the product) should be distributed through appropriate channels!
    • Org-mode blog post
    • YouTube
  6. It should be indefinitely revised and updated over time!

Litera.cy

I want to write a tool that sort of… extends org-mode as a literate programming/writing environment and is also a play-writing-esque scripting language! The tool then can convert an org file into a video presentation! This will then power my YouTube channel! Basically I can add “literate documentation” to all of my projects, which gets automatically mirrored in text form onto the Tator Tot Club as well as to my YouTube channel. The “literate part” is sort of the meta project, and it produces a “behind the scenes” piece of content. So for every content I produce, I get a “meta-content” for free!

Powered by little girl simulator!

Chamomile OS

I want to build an “operating system” that encompasses the following ideas:

  • Lisp OS inspired
  • Emacs inspired
  • Org-mode inspired
  • Zettelkasten inspired
  • Super-memo inspired
  • Anki inspired
  • Self-tracking features
  • Probabilistic shell language with inspiration from Lisp and languages like Idris

Essentially, it’s a certain kind of environment that aims to sort of maximize and generalize the idea of a computer as an extension of yourself. It doesn’t have to strictly be implemented as a standalone operating system, but that would be very satisfying! In the mean time, I use Emacs and configure it as closely to this goal as possible. (And so far it’s very functionable!)

OmniLingo

A social website aimed at language learning! With inspiration from Lang-8, but using a more generalized diff-based and version controlled wiki system for community post edits. And with badges and rewards and a currency system with inspiration from an old version of RosaSecta (old defunct private art site).

It should also be centered around learning via parallel-media with many languages in parallel. Spaced repetition tools similar to Anki+subs2srs. Inspiration also from LWT (Learning With Texts) and LingQ.
